# Secure Login System (Frontend Demo)

This project demonstrates a simple login system built with HTML, CSS, and JavaScript, focusing on secure coding practices.

## Features
- Input validation (email & password)
- Basic password rules
- Safe DOM manipulation (prevents XSS)
- User feedback handling

## Note
This is a frontend-only demo. In real-world applications, authentication and security must be handled on the backend.
